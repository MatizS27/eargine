# Eargine

**Middle Ear Pressure Amplification Effect Simulator**

Eargine is a first-year university project that simulates the middle ear pressure amplification effect and integrates with Arduino hardware. The project combines basic auditory modeling with physical interaction using sensors, aiming to explore how pressure variations can influence sound perception.
